Exercise 3.

Recall that the power set of any set \(A\) is the set of all subsets of \(A\) and is denoted \(\mathcal{P}(A)\text{.}\) Which of the following are true?

\begin{gather*} \mathcal{P}(A \cap B) = \mathcal{P}(A) \cap \mathcal{P}(B)\\ \mathcal{P}(A \cup B) = \mathcal{P}(A) \cup \mathcal{P}(B) \end{gather*}

If either is not true, can you replace the equals sign with \(\subseteq\) or \(\supseteq\) to get a true statement?
