Exercise 4.

Let \(A\) be the set of strings of 0's and 1's of length 3 or less; and let \(B\) be the set of strings of 0's and 1's of length 3. What properties do the following relations have?

  1. Define the relation of \(w\) on \(A\) by \(x w y\) if \(x\) has the same number of 1's as \(y\text{.}\) For example, \(01 w 100\text{,}\) but \(01 w 101\) is false.
  2. Define the the relation \(d\) on \(B\) defined by \(x d y\) if \(x\) differs from \(y\) in exactly one position. For example, \(100 d 101\text{,}\) but \(100 d 111\) is false.
  3. Define the the relation \(c\) defined on \(A\) by \(x c y\) if \(x\) is contained within \(y\text{.}\) For example, \(10 c 101\text{,}\) but \(11 c 101\) is false.

For any of these relations that are partial orderings, draw the Hasse diagram for that relation. For any of them that is an equivalence relation, identify the equivalence classes.
