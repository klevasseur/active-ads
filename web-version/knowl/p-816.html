  1. Suppose a two bit message is encoded into a five bit message using the function \(e(b_{1}b_{2})=b_{1}b_{1}(b_{1}+_2 b_{2})b_{2}b_{2}\text{.}\) What matrix, \(G\text{,}\) has the property that \(e(b_{1}b_{2})=(b_{1} b_{2}) G\text{?}\)

  2. Define the two functions \(g:\mathbb{Z}_2{}^3\rightarrow \mathbb{Z}_2{}^4\) and \(p:\mathbb{Z}_2{}^4\to \mathbb{Z}_2\) by \(g(a_1,a_2,a_3) = (a_1,a_2,a_3 ,a_1+_2 a_2+_2a_3)\text{,}\) and \(p(b_1,b_2,b_3,b_4)=b_1+b_2+b_3+b_4\text{.}\)

    1. The values of \(g(a_1,a_2,a_3)\) and \(p(b_1,b_2,b_3,b_4)\) can be computed as matrix products. Determine the matrices.

    2. Describe the function \(p \circ g\text{.}\) Is it a homomorphism?

  3. Suppose we were to send three bits across a binary symmetric channel by first encoding the bits using the function \(g\) from the previous problem, producing a sequence of four bits. After receiving the four bits, the person at the receiving end applies the function \(p\) to those bits. What can we derivive form the output of \(p\text{?}\)

  4. Linear Codes. Let \(m\) and \(n\) be positive integers. Suppose that \(G\) is an \(m \times n\) matrix of zeros and ones. Then we can can define a function \(e:\mathbb{Z}_2^m \rightarrow \mathbb{Z}_2^n\) by \(e(a)=a G\) where we take \(a \in \mathbb{Z}_2^m\) to be a \(1\times m\) matrix and we perform mod 2 arithmetic in computing the product \(a G\text{.}\) Prove that \(e\) is a homomorphism.
