Reading Question 3.

Let \(A = \{0, 2, 3\}\text{,}\) \(B = \{2, 3\}\text{,}\) \(C = \{1, 5, 9\}\text{,}\) \(D = \{3, 2\}\text{,}\) and \(E = \{2, 3, 2\}\text{.}\) Assume that the universal set is \(U = \{0, 1, 2, . . . , 9\}\text{.}\) Determine which of the following are true. Give reasons for your decisions.

  1. \(\displaystyle A = B\)

  2. \(\displaystyle B = C\)

  3. \(\displaystyle B = D\)

  4. \(\displaystyle E=D\)

  5. \(\displaystyle A\cap B = B\cap A\)

  6. \(\displaystyle A \cup B = B \cup A\)

  7. \(\displaystyle A-B = B-A\)

  8. \(\displaystyle A \oplus B = B \oplus A\)
