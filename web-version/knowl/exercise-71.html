Exercise 1.

A common way to denote a particular minset generated by a collection of subsets is as follows. If there are \(k\) subsets, \(B_1, B_2, \dots ,B_k\text{,}\) and \(b=b_1b_2\cdots b_k\) is any string of \(k\) bits, then

\begin{equation*} M_b = M_{b_1b_2\cdots b_k} = D_1 \cap D_2 \cap \cdots \cap D_k, \end{equation*}

where \(D_i\) is either \(B_i\) or \(B_i^c\text{.}\) If \(b_i = 1\) then \(D_i =B_i\) and if \(b_i=0\) then \(D_i=B_i^c\text{.}\) For example, if \(k=4\text{,}\) \(M_{0110} = B_1^c \cap B_2 \cap B_3 \cap B_4^c\text{.}\)

  1. Suppose \(U=\{1, 2, 3, 4, 5\}\text{,}\) \(k=2\text{,}\) \(B_1= \{1, 2\}\text{,}\) and \(B_2 = \{2,3,4\}\text{.}\) List the nonempty minsets generated by \(B_1\) and \(B_2\) using “\(M_b\)” notation. Notice that they form a partition of \(U\text{.}\)

  2. How does this notation make help us see how many distinct nonempty minsets there could be that are generated by \(k\) subsets of a universe.
