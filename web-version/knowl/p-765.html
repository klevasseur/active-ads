Read Section 15.2 of Applied Discrete Structures. If you get confused in this section, think about the partition of the integers into even integers and odd integers. There is a consistency in doing arithmetic on the integers where, for example, any time you add an even integer and an odd integer, you get an odd integer. So \(\textrm{even} + \textrm{odd} = \textrm{even}\text{.}\) This is coset addition! For any group, the only way something like this can work is when the partition is into cosets, which is why we start by defining this set of subsets of a group.
