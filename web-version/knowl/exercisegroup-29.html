1.

Let \(A = \{a, b, c, d\}\text{.}\) Draw the graphs of relations on \(A\) where:

  1. The first relation is reflexive, symmetric, but not transitive.

  2. The second relation is transitive, but not symmetric and not reflexive.

  3. The third relation is both an equivalence relation and a partial ordering.

2.

Let \(A = \{0, 1, 2, 3\}\) and let

\begin{equation*} r = \{(0, 0), (1, 1), (2, 2), (3, 3), (1, 2), (2, 1), (3, 0), (0, 3)\} \end{equation*}

  1. Verify that \(r\) is an equivalence relation on \(A\text{.}\)

  2. Let \(a \in A\) and define \(c(a) = \{b \in A \mid a rb\}\text{.}\) \(c(a)\) is called the equivalence class of \(a\) under \(r\). Find \(c(a)\) for each element \(a \in A\text{.}\)

  3. Show that \(\{c(a) \mid a \in A\}\) forms a partition of \(A\) for this set \(A\text{.}\)

  4. Let \(r\) be an equivalence relation on an arbitrary set \(A\text{.}\) Prove that the set of all equivalence classes under \(r\) constitutes a partition of \(A\text{.}\)

3.

Describe the equivalence classes under the relation congruence modulo 10 on the integers.

4.

Let \(A\) be the set of strings of 0's and 1's of length 3 or less; and let \(B\) be the set of strings of 0's and 1's of length 3. What properties do the following relations have?

  1. Define the relation of \(w\) on \(A\) by \(x w y\) if \(x\) has the same number of 1's as \(y\text{.}\) For example, \(01 w 100\text{,}\) but \(01 w 101\) is false.
  2. Define the the relation \(d\) on \(B\) defined by \(x d y\) if \(x\) differs from \(y\) in exactly one position. For example, \(100 d 101\text{,}\) but \(100 d 111\) is false.
  3. Define the the relation \(c\) defined on \(A\) by \(x c y\) if \(x\) is contained within \(y\text{.}\) For example, \(10 c 101\text{,}\) but \(11 c 101\) is false.

For any of these relations that are partial orderings, draw the Hasse diagram for that relation. For any of them that is an equivalence relation, identify the equivalence classes.
